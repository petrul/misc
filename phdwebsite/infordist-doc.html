Documentation pour Infordist GUI

Infordist GUI est une application web qui permet d'inspecter des distances sémantiques à base de NGD : http://www.infres.enst.fr/searcher_app/

Rappel, étant donnés deux terms x et y, la fréquence d'apparition dans le corpus détermine g(x) = nr_hits(x)/nr_total_documents, assimilable à une probabilité. Le code attendu G(x) = -log g(x) donne la complexité du terme.

Etant donné G(x,y) et G(x), on peut calculer G(y|x) = G(x,y) - G(x). Exact formulas for NGD, are here.

Quatre colonnes, donc:
  • NGD: Normalized Google Distance: max( G(x|y), G(y|x) ) / max(G(x), G(y))
  • UNGD: unnormalized NGD: max( G(x|y), G(y|x) )
  • The last two columns are conditional complexities, G(x|y) and G(y|x). They are no longer symmetrical, hence two columns, not one
